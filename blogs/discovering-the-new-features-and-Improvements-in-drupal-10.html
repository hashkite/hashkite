Introduction

Drupal is in a constant state of evolution, meeting the demands of web application development. As a leading open source CMS, Drupal is one of the leading choices for building web applications. With the recent transition from Drupal 9 to Drupal 10, read the following pointers to stay informed about all the significant Drupal 10 features.

Drupal 10 Release

Drupal 10 made its debut on December 14, 2022 adhering to the community's tradition of unveiling major updates on Wednesdays. For those eager to explore Drupal 10, a beta release is also available for experimentation.

Drupal 10 Features

Drupal 10 introduces several notable features, including:

Olivero Default Theme

One of the prominent upgrades in Drupal 10 is the inclusion of the Olivero default theme. This theme, which has been in development for some time, significantly enhances the visual appearance of websites. Compared to previous core themes, Olivero offers a more modern and visually appealing design.

With Olivero, developers no longer need to create themes from scratch for their websites. This default theme serves as an excellent starting point for creating custom themes. Additionally, for headless projects, Olivero allows developers to preview staged content in a structured manner before integrating it into the final application.

The inclusion of Olivero as the default theme not only improves the initial impression of Drupal websites but also serves as a tribute to the former community member Rachel Olivero.

Claro Administration Theme

The administrative and editorial experience is crucial for any CMS. For a long time, Drupal relied on the clean administrative theme 'Seven,' which served most users well and integrated nicely with various contributed modules.

However, Seven's compatibility with recent improvements, particularly the introduction of modern JavaScript concepts, became a concern. This led community members to develop a new administrative theme with a highly accessible user interface: 'Claro.'

Claro enhances the administrative experience and provides Drupal's editorial UI with a modern feel while preserving the core functionality of the CMS to prevent confusion among experienced users.

Starting with Drupal 10, Claro will be the default admin theme and will receive updates bundled with Drupal core releases. For a detailed comparison between Seven and Claro, refer to our comprehensive guide.

CKEditor 5

CKEditor 5 has established itself as a prominent tool for managing text data, undergoing a comprehensive overhaul to align with modern JavaScript application development practices. This enhancement has been widely adopted across various frameworks and CMS platforms, significantly improving the editorial experience for content management.

The successful integration of CKEditor 5 with Drupal has led to its adoption as the default editor in Drupal 10, ensuring a seamless text editing experience. Many CKEditor plugins available on Drupal.org have been upgraded to be compatible with CKEditor 5.

Modern JavaScript Components To Replace Some Uses Of jQuery

The JavaScript development community is increasingly embracing ES6 standards and favoring pure JavaScript over jQuery for tasks such as DOM manipulation and AJAX requests. This shift is driven by factors including the integration of jQuery-like functionality into web browsers and the popularity of advanced frameworks like React and Vue.js.

The Drupal community has closely observed this trend and initiated the replacement of jQuery with modern JavaScript throughout the core codebase. In Drupal 10, jQuery libraries like jQuery.once and jQuery.cookie have been completely removed, and many contributed modules are transitioning away from jQuery and its libraries. This results in a cleaner front-end output for Drupal websites.

Theme Starterkit Tools For Bespoke Theme Creation

Theming is a crucial aspect of Drupal development, as it determines the appearance of an application. To make theming easier, more efficient, and aligned with the latest trends, the Drupal community has introduced the Theme Starterkit for Drupal 10. This tool provides developers with commands to effortlessly generate a theme with all the necessary files and configurations.

Symfony 6

Drupal 10 will adopt Symfony 6 as its foundation, requiring a minimum of PHP 8.1 support. This strategic decision ensures that the backend processing enhancements offered by Symfony 6 are seamlessly integrated into all Drupal projects upgrading to Drupal 10.

Symfony 6 brings significant improvements in areas such as routing, services, locales, and multilingual support. These enhancements will gradually be considered for integration into Drupal.

Be Drupal 10 Ready

To get ready for Drupal 10, follow these essential steps:

  1. Update From Drupal 9: If you're currently using Drupal 9, ensure that you update the core codebase and all contributed modules to their latest releases. This is crucial for a smooth transition to Drupal 10.
  2. Drupal 8 Upgrade: Drupal 8 reached its end of life in November 2021. If you're still on Drupal 8, initiate the upgrade process to Drupal 9 as soon as possible. Many vital contributed modules have already released versions compatible with Drupal 10.
  3. PHP 8.1: Make the switch to PHP 8.1, as it is the minimum supported version for Drupal 10. Detailed PHP-related requirements can be found here.
  4. Identify Deprecated Code: Utilize the Upgrade Status module to identify deprecated code in your custom modules and start rectifying them. The Drupal Rector tool can expedite this task.
  5. Test And Plan: Create a new instance and upgrade the Drupal core to a 10-beta release for testing and planning purposes. This provides developers and testers with the necessary resources for a smooth transition to Drupal 10.

Exciting Features Lined Up For Drupal 10.1 And Further Releases

Automatic Updates

One of the upcoming features is the Automatic Updates module, which simplifies the process of applying critical Drupal core updates. It automates the update process, generates a sandbox instance, and deploys necessary patches, all without the need for extensive development efforts. The module also offers detailed progress reporting and an API for customization.

Project Browser

The Project Browser streamlines the search for contributed modules and themes on Drupal.org. Administrators can conveniently browse, access essential information, and even install these modules directly from the website's user interface. This approach provides a more user-friendly alternative to the previous module search and installation methods.

The Conclusion

Drupal 10 introduces a diverse range of features and improvements. It actively addresses bug fixes and feature requests on a daily basis, ensuring their integration into the inaugural Drupal 10 release. This release significantly enhances existing websites and presents compelling reasons for users of other Drupal versions to transition to Drupal 10.

If you're a developer, be sure to explore the straightforward upgrade procedures to Drupal 10 and stay informed about the progress of contributed projects.
