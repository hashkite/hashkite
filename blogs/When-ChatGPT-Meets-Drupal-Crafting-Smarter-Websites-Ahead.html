Using the Drupal OpenAI/ChatGPT/AI Search Module

This module, available as a contributed extension for Drupal, merges the capabilities of ChatGPT into the Drupal ecosystem. Currently, it's designed for use with Drupal 10 and is in an alpha release phase. You can install it using Composer.

composer require 'drupal/openai:^1.0@alpha'

The module encompasses nine submodules, each offering distinct functionalities. Some of these include:

  • ChatGPT/Prompt Explorer
  • Content Tools
  • CKEditor integration
  • Log analyzer
  • Text Embeddings

This blog will focus on functionalities related to content generation and manipulation.


1. Setting up
→ Create an account on the OpenAI website.
→ Generate a secret key.
→ Note down the Organization ID.
→ Enter these values in the OpenAI settings within Drupal's admin panel.

Image: OpenAI settings interface in Drupal setup

2. ChatGPT/Prompt Explorer
→ Allows administrators to experiment with OpenAI services.
→ Functions like ChatGPT, where you ask a question, and it responds.
→ Offers options like model selection, max tokens, and temperature for varied responses.

GIF: ChatGPT/Prompt Explorer

3. CKEditor
→ Integrates ChatGPT into CKEditor for content generation.
→ Includes options for summarizing, translating, adjusting tone, and correcting HTML.
→ Each function sends a question to ChatGPT and receives a response.

Image: CKEditor settings from article content type


4. Content Tools
→ Similar to CKEditor but applicable to all entity forms.
→ Allows content generation using any field.

Image: Content Tools options on node form.


5. Log Analyzer
→ Adds explanatory messages to Drupal's logging system for easier understanding.
→ Simplifies error messages.

Image: Logger Analysis
